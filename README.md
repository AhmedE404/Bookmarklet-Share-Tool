# 🚀 Bookmarklet Forge

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Vanilla JS](https://img.shields.io/badge/JavaScript-Vanilla-yellow.svg)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

A modern, zero-dependency, client-side application that allows developers to write, compress, and share JavaScript bookmarklets easily via secure, compressed URLs.

## 🤔 What is a Bookmarklet?
A bookmarklet is a small JavaScript program stored as a URL within a bookmark in your web browser. Instead of navigating to a website, clicking a bookmarklet executes the code directly on the current page. It's perfect for web scraping, DOM manipulation, or automating repetitive tasks.

## ✨ Features
* **Zero Dependencies:** Built entirely with Vanilla JavaScript and HTML/CSS. No external libraries or CDNs required.
* **Native Compression:** Uses the modern `Compression Streams API` (`deflate-raw`) to compress code, ensuring shareable URLs remain short and never hit server length limits.
* **100% Client-Side:** Secure by design. Your code is compressed and processed directly in the browser. No data is sent to any server.
* **Smart UI/UX:** * Automatically switches between "Creation Mode" and "Installation Mode" based on the URL Hash.
  * Drag-and-drop installation interface.
  * Automatically detects macOS vs. Windows to display the correct keyboard shortcuts (⌘ vs. Ctrl).

## 🛠️ How to Use

### 1. Create & Share a Bookmarklet
1. Open the [Bookmarklet Forge](https://AhmedE404.github.io/Bookmarklet-Share-Tool) hosted page.
2. Enter your tool's name and paste your JavaScript code.
3. Click **"Generate Compressed Link"**.
4. Copy the generated URL and share it with anyone!

### 2. Install a Shared Bookmarklet
1. Open a shared link generated by this tool.
2. You will see a minimalist installation screen.
3. Simply drag the **"Drag Me ≡"** button to your browser's Bookmarks Bar.
4. If your Bookmarks Bar is hidden, use `Ctrl + Shift + B` (or `⌘ + Shift + B` on Mac) to show it.
